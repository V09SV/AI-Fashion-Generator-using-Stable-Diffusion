# 👗 AI Fashion Generator using Stable Diffusion

## 📌 Overview
This project is an AI-powered fashion image generator that creates clothing designs from text prompts using deep learning techniques. It leverages Stable Diffusion to generate realistic and creative fashion outputs.

---

## 🚀 Features
- Text-to-image fashion generation
- User-friendly UI
- AI-powered model integration
- Real-time image generation
- Custom prompt-based design creation

---

## 🛠️ Tech Stack
- Python
- Flask
- Stable Diffusion
- PyTorch
- HTML/CSS/JavaScript
